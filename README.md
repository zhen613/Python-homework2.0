# Python-homework
# 從土地到餐桌：田頭水的食農教育——Python基礎教材

## 專案簡介
這是為大學社會責任（USR）計畫設計的Python基礎教材，主題為「從土地到餐桌：田頭水的食農教育」。課程聚焦農業灌溉中水井的重要性，透過台灣真實水井資料（來自水資源署及相關研究），讓初學者使用Python計算水井可用水深，模擬農田灌溉情境。教材僅使用`print`、`input`、變數及運算子（+、-、*、/），適合零基礎學習者，無需分支、迴圈、函式或類別。

## 檔案結構
- `example1.py`: 固定值計算——桃園華興站井No.1（井深25公尺，水位深度6.7公尺）
- `example2.py`: 使用者輸入計算——桃園華興站井No.2（井深110公尺）
- `example3.py`: 多變數計算與水量估計——桃園華興站井No.3（井深180公尺）
- `README.md`: 本文件，介紹專案與使用說明
